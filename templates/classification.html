Classification in Machine Learning

Classification is a type of supervised learning in machine learning where the algorithm is trained to categorize data into predefined classes.

The goal of classification is to predict the class of an unseen instance based on the characteristics of the instances in the training dataset.

Types of Classification

  1. Binary Classification:

    Binary classification problem is one where the algorithm has to predict between two classes. For example, email spam classification.

  2. Multiclass Classification

    A multiclass classification problem is one where the algorithm has to predict between multiple classes. For example, image classification where the image can belong to any one of several classes such as dogs, cats, birds, etc.

  3. Multi-Label Classification

    A multi-label classification problem is one where an instance can belong to more than one class. For example, a news article can be categorized under both politics and sports

Here are some commonly used models for different types of classification

1. Binary Classification

  1. Logistic Regression
  2. Support Vector Machines(SVM)
  3. k-Nearest Neighbor(k-NN)
  4. Naive Bayes
  5. Decision Trees
  6. Random Forest

2. Multiclass Classification

  1. One-vs-All(One-vs-Rest)
  2. Multinomial Logistic Regression
  3. Decision Trees
  4. Random Forest
  5. k-Nearest Neighbor(k-NN)

3. Multi-label Classification

  1. Binary Relevance
  2. Classifier Chains
  3. Label Powerset
  4. Random Forest-based models
